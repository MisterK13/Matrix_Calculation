# Калькулятор матриц
Калькулятор матриц с графическим интерфейсом на PyQt5. Поддерживает базовые операции с матрицами размером до 3×3.

## Возможности
✨ Сложение матриц

✨ Вычитание матриц

✨ Умножение матриц

✨ Вычисление детерминанта (для матриц 1×1, 2×2 и 3×3)

## Особенности
Интуитивно понятный интерфейс

Динамическое добавление строк и столбцов (до 3×3)

Автоматическая очистка пустых строк и столбцов

Валидация вводимых данных (только числа, не нули)

Поддержка отрицательных чисел

Адаптивный интерфейс под выбранную операцию
